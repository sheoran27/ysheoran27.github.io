Detailed User Guide

Say the wakeword followed with one of the query specific keywords

New Query specific Keywords-

  1. time - current time
  2. date - current date
  3. email - send email
  4. message - send message
  5. wikipedia - search on wikipedia
  6. search - search on web
  7. youtube - play on youtube
  8. weather - current weather
  9. news - listen news
  10. read - convert text to speech
  11. covid - listen covid updates
  12. joke - listen some joke
  13. screenshot - take a screenshot
  14. take/write note - take down some notes
  15. do i have pending tasks - 3 recent pending tasks

Fetch Query specific Keywords-

  1. get emails - recent emails
  2. get messages - recent messages
  3. get wikipedia - recent wikipedia searches
  4. get google search - recent google searches
  5. get youtube favourites - youtube favourite videos
  6. get weather - recent weather searches
  7. get news - recent searched news
  8. get websites - recent accessed websites
  9. get pending notes - get your pending notes
  10. get notes - get your recent notes
